# Store
## Requirements

- Node.js
## Libs

- ReactJS
- Styled Components
- Redux
- Redux Thunk
- React Redux
- React Loading 

## Installation

1. Run command on the root folder

```bash
$  yarn install 
```
## Running

To run the app

```bash
$  yarn start
```
